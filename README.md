# 中文词组翻译器 - Chrome扩展

一个功能强大的Chrome浏览器扩展，能够自动识别网页中的中文词组并使用翻译API进行翻译，帮助用户更好地理解外文内容。

## 主要功能

### 🔍 智能识别
- 自动检测网页中的中文词组（2个字符以上）
- 使用正则表达式精确匹配中文字符
- 避免重复处理已翻译的内容

### 🌐 多语言翻译
- 支持多种目标语言：英语、日语、韩语、法语、德语、西班牙语、俄语
- 使用Google翻译API，翻译质量可靠
- 可自定义翻译延迟，避免API限制

### 💾 实时翻译
- 实时翻译中文词组，无需等待
- 支持多种翻译服务
- 本地存储配置，保护用户隐私

### ⚙️ 灵活配置
- 可自定义翻译格式：[中文|翻译]、(中文|翻译)、中文: 翻译
- 可调整翻译延迟时间
- 支持暂停/恢复翻译功能

### 📊 状态信息
- 实时显示翻译状态
- 支持暂停/恢复功能
- 实时更新状态信息

## 安装方法

### 方法1：开发者模式安装
1. 下载或克隆本项目到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹，完成安装

### 方法2：打包安装
1. 在扩展管理页面点击"打包扩展程序"
2. 选择项目根目录，生成.crx文件
3. 将.crx文件拖拽到扩展管理页面进行安装

## 使用方法

### 基本使用
1. 安装扩展后，在任意网页上点击扩展图标
2. 扩展会自动检测页面中的中文词组
3. 翻译结果会以 `[中文|翻译]` 的格式显示在原文旁边

### 高级设置
1. 点击扩展图标，选择"打开设置"
2. 在设置页面可以：
   - 选择目标语言
   - 调整翻译延迟
   - 选择翻译格式
   - 配置翻译服务

### 快捷键操作
- 点击扩展图标：打开弹窗界面
- 右键扩展图标：快速访问设置

## 技术架构

### 文件结构
```
chrome_ext/
├── manifest.json          # 扩展配置文件
├── scripts/
│   ├── content.js        # 内容脚本，处理页面翻译
│   └── background.js     # 后台服务脚本
├── popup/
│   ├── popup.html        # 弹窗界面
│   ├── popup.js          # 弹窗脚本
│   ├── options.html      # 设置页面
│   ├── options.js        # 设置脚本
│   └── button.css        # 样式文件
├── images/
│   └── icon.png          # 扩展图标
└── README.md             # 项目说明
```

### 核心组件
- **Content Script**: 负责页面文本检测和翻译处理
- **Background Script**: 管理扩展生命周期和统计信息
- **Popup**: 提供用户交互界面
- **Options Page**: 提供详细配置选项

### 技术特点
- 使用Chrome Extension Manifest V3
- 支持动态内容检测（MutationObserver）
- 异步翻译处理，不阻塞页面渲染
- 本地存储和同步存储结合使用

## 配置选项

### 翻译设置
- **目标语言**: 选择翻译的目标语言
- **翻译延迟**: 设置翻译请求的延迟时间（毫秒）
- **翻译格式**: 选择翻译结果的显示格式

### 翻译服务
- **Google翻译**: 默认翻译服务，免费且稳定
- **有道云翻译**: 需要配置API密钥
- **阿里云翻译**: 需要配置API密钥

## 注意事项

### API限制
- Google翻译API有请求频率限制
- 建议适当调整翻译延迟参数
- 大量翻译时建议分批处理

### 隐私保护
- 翻译请求直接发送到选择的翻译服务
- 本地仅存储配置信息
- 可随时清除配置数据

### 兼容性
- 支持Chrome 88+版本
- 支持基于Chromium的浏览器（Edge、Opera等）
- 不支持Firefox（需要适配）

## 常见问题

### Q: 为什么有些中文没有被翻译？
A: 可能的原因：
- 文本节点已被处理过
- 包含在script、style等标签中
- 翻译API请求失败

### Q: 如何提高翻译速度？
A: 建议：
- 减少翻译延迟设置
- 选择合适的翻译服务
- 避免同时翻译大量内容

### Q: 翻译结果不准确怎么办？
A: 解决方案：
- 尝试不同的翻译服务
- 调整目标语言设置
- 检查网络连接状态
